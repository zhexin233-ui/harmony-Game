# CHANGELOG

## v1.0.0 — 2026-04-19

首发版本。

### 功能
- 5 款完整可玩小游戏：摇一摇赛马 / 定时炸弹 / 鳄鱼拔牙 / 同屏反应大比拼 / 指尖大轮盘。
- 本地惩罚库：25 条内置 + 用户自定义增删改 + 启停开关。
- 双主题：Q 版卡通 / 霓虹电玩，首次启动引导，设置内切换。
- 音效 / 振动 / 屏幕常亮 / 单机共玩交互。
- 完整本地持久化（settings · punishments · hasOnboarded · lastSession）。

### 兼容性
- HarmonyOS NEXT 6.x
- 未适配 iOS / Android / H5 / 小程序（留有架构口）。
